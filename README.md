# 🌱 Aranyak - Plant Landing Page

Aranyak is a modern and elegant landing page template for plant stores and eco-friendly businesses. It features a clean design, responsive layout, and beautiful sections for showcasing products, team members, and company philosophy.

---

## 📌 Features

- 🌿 Hero section with CTA
- 🛒 Product catalog section
- 👥 About us and team section
- 📈 Animated counters (plants delivered, happy customers, etc.)
- 🔍 Search bar
- 📱 Responsive design (mobile-friendly)
- 🔤 RTL support
- 🎨 Stylish typography and icons

---

## 🧰 Built With

- HTML5
- CSS3
- Bootstrap 5
- Swiper.js
- Unicons
- Gulp (optional)
